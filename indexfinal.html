Data Structures and Algorithms

In this tutoral you are going to learn about Data Structures concepts effectively with examples This course will be a general overview of data structures and so we won’t be confining ourselves to one specific language

what exactly is a data structure?

Well in Computer Science, a data structure is a way to store, organize, and manage data -or information in a way that allows you the programmer to easily access or modify the values within them.

Types of Data Structures

  1. Arrays
  2. ArrayLists
  3. Stack
  4. Queue
  5. Maps

Which data structure we have to choose for a problem?

So the most common functions you might want from a data structure. Are The ability to access a specific element within the data structure, search for a particular element within the data structure, insertan element into the data structure, and remove an element from the data structure. Now all datastructures provide these features in different ways,But which data structure is useful for your problem is depend upon problem or user requrements as well as effeciency or perfamence datastructure for example, we might choose a data structure which can access elements the quickest. Vice versa, if accessing elements isn’t the most important thing to us, but we need a data structure which can be easily added to and deleted from, we would go for one which is most efficient in that specific functionality.

Effeciency of a datastructure?

The efficiency of an algorithm depends upon its time complexity and space complexity

For finding efficiency we use Big O notation
